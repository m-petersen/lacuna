## What You'll Learn

By completing these tutorials, you will:

1. **Install Lacuna** and verify your environment is set up correctly
2. **Load lesion data** from NIfTI files
3. **Run a functional LNM analysis** and understand the output
4. **Save and export results** in standard formats

## Prerequisites

- Python 3.10 or higher
- Basic familiarity with the command line
- Basic Python knowledge

## Time Estimate

Each tutorial takes approximately 15-20 minutes to complete.
